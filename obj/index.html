Lidar tests

The idea of this set of notes is to allow the reader to gain some experience of using the software start for lidar simulation. It is intended to run these exercises in the start/tests/lidar directory of your librat distribution (although it should run independently as well on most systems).

If you do not already have a distribution of librat, get one here.

You can download all of the files you will need for these examples from here. Contact Professor P. Lewis if you have any questions.
