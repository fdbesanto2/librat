Monte Carlo ray tracing library librat

Prof. P. Lewis, UCL Geography
To download and install, download and run this configure script. It should work on any unix-like operating system (tested on Solaris, various linux, Mac OS X, etc.).

Example 3D animations
This page contains a series of animations of both reflectance and lidar simulations, as animated gifs (large) and QT movies (smaller).
